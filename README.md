# IT-Sensing-Prototype

Pembuatan aplikasi Desktop berdasarkan website IT Sensing

✅ CHECKLIST FITUR YANG SUDAH DIBUAT :

* Layout UI (Belum Ditambah UI) ✔
* Toolbar kanan Peta (Sudah ditambah UI) ✔
* Peta menggunakan Tiles download ✔
* Scroll dan Drag di Peta ✔
* Map Style Switching ✔
* Script Proxy Cloudflare dan No Proxy (Jika misal peta tidak load/block connection) ✔

✅ CHECKLIST LAPORAN DEBUGGING

* UI Peta (drag and scroll) hanya bisa di area Peta
* Tile direfresh saat zoom atau drag
* Pergerakan Zoom dan Drag lebih smooth
* Zoom range dibatasi (2–19)
* Untuk switching style Peta Semua tile lama dihancurkan
* Style "Terrarin" Masih Belum bisa diakses  ⚠️
* Untuk Mengatasi jika gambar tidak bisa load kemungkinan karena di block dari google maka menggunakan proxy Cloudflare ⚠️
